<defs> Demo — Self-contained SVG examples

No external images used. Each card shows a different use of <defs>: gradient, clipPath, mask, <use>/symbol. Below each SVG you’ll find the concise code snippet used.

1. Linear Gradient (reusable)

<defs>
  <linearGradient id="g1Gradient"> ... </linearGradient>
</defs>

<rect fill="url(#g1Gradient)" />

2. clipPath — constrain drawing area

<defs>
  <clipPath id="circleClip">
    <circle cx="120" cy="110" r="90"/>
  </clipPath>
</defs>

<g clip-path="url(#circleClip)"> ... </g>

3. mask — transparency map (fade)

<linearGradient id="maskGrad"> ... </linearGradient>
<mask id="leftFade">
  <rect fill="url(#maskGrad)"/>
</mask>

<rect mask="url(#leftFade)" />

4. <symbol> + <use> — define once, place many times

<symbol id="chip"> ... </symbol>
<use href="#chip" x="30" y="30" width="120" height="72" />

This demo is self-contained and uses only SVG shapes and gradients — no external images or network resources. You can save the page as defs-demo.html and open it locally in any modern browser.
